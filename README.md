# fable-expo-template
Expo template for using F#/Fable with React Native

# Prereqs

.NET Core >= 5

node >= 16

Expo CLI

# Quickstart

`$ git clone https://github.com/bobcat-pixel/fable-expo-template && cd fable-expo-template`

`$ npm install`

`$ dotnet fable watch ./src -o ./out`

`$ npm start`
